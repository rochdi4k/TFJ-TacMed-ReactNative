# Code of Conduct

## Our Pledge

We are committed to making this project a respectful, safe, and professional environment for everyone.

This project is focused on building and improving TFJ TacMed React Native, a tactical medication calculator and reference tool. Contributors are expected to act responsibly, communicate clearly, and respect the safety-critical nature of the application.

## Expected Behavior

Contributors are expected to:

- Be respectful and professional.
- Use clear and constructive language.
- Accept feedback gracefully.
- Keep discussions focused on improving the project.
- Report bugs and issues honestly.
- Avoid making unsupported medical claims.
- Respect the fact that this project may involve medical-reference calculations.

## Unacceptable Behavior

The following behavior is not acceptable:

- Harassment, threats, insults, or personal attacks.
- Discrimination or hateful language.
- Trolling, spam, or disruptive behavior.
- Publishing private information without permission.
- Submitting intentionally misleading code, data, or documentation.
- Promoting unsafe medical use without protocol verification.
- Presenting this app as a replacement for authorized medical guidance.

## Medical Safety Conduct

Because this project involves medication calculations, contributors must not:

- Add medication values without a clear source.
- Modify dose logic without explaining the reason.
- Remove safety warnings.
- Claim the app is clinically validated unless formal validation exists.
- Encourage real-world use without official protocol review.

All medication data must be verified against authorized medical protocols before real use.

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributions that violate this Code of Conduct.

Maintainers may also block users who repeatedly violate these rules.

## Reporting

To report a problem, open a GitHub issue or contact the repository owner privately.

When reporting, include:

- What happened
- Where it happened
- Relevant screenshots or links
- Any safety concerns

## Scope

This Code of Conduct applies to:

- GitHub issues
- Pull requests
- Discussions
- Documentation
- Code comments
- Any project-related communication

## Attribution

This Code of Conduct is inspired by common open-source community standards and adapted for a medical-reference software project.
